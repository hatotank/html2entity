# html2entity
メールアドレスなどをHTMLエンティティに変換
- [ダウンロード](https://github.com/hatotank/html2entity/releases)

## 概要
ＨＴＭＬ文章のリンクやメールアドレスをエンティティに変換するプログラムです。  
過去に作成したVB6プログラムを.NETに変換。
